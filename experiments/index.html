Animated Big Data Visualization

Experiments

This is the overview page for all available experiments. All of these are based on D3.js. The first two make use of its Force-Directed Layout Algorithm or a related concept. The later two use it as an SVG/Visualization library but implement their own Layout Algorithm.

  1. Animated Force-Directed Layout
  2. Multiple Focal Points
  3. Horizontally Centered Layout
  4. Vertically Centered Layout

The Animated Force-Directed Layout proves the possibility of a chronological animation with the chosen toolset, but has layout issues. Multiple Focal Points is an approach to solve these layout issues by implementing a custom variation of the force-directed layout. It works quite well in theory, but provides no collision detection.

Horizontally Centered Layout solves the layout issue better. The graph is stringently structured and reduces crossings and collisions, but is not scalable. Vertically Centered Layout attempts to solve the scalability issue by rotating the hypergraph by 90Â° and is the final experiment.
