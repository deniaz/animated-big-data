Horizontally Centered Layout

Graph Layout Algorithm without the use of Force Layout, which assigns all items centrally

This experiment tries to solve the issues overlapping links and nodes. The code is still using D3, but only as some kind of view library rather than a graph visualization. The use of D3's Force Layout was entirely dropped.

This approach aligns all items horizontally centered. The frequency nodes are positioned above or below the attributes. As shown above this reduces overlapping of both links and nodes drastically. User Interaction is still possible through draggable nodes. Thus a user can re-draw a graph by dragging nodes to another position.

This graph layout was tested in a Usability Audit. Some of the feedback from the audit was implemented and resulted in the following experiment Vertically Centered Layout.
