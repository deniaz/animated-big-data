D3 Timeline Experiment

Timeline data in Subgraph model

In this experiment the approach of storing the timeline data within the subgraph model was tested.

In this experiment nodes are not sorted. They are injected into the graph just as they are in the JSON file.

For this experiment some variations were built with sorting of the nodes. See the following links:

  • Most connected nodes first
  • Least connected nodes first
  • Frequencies first, than most connected attributes
  • Frequencies first, than least connected attributes
  • Most connected attributes first, than frequencies
  • Least connected attributes first, than frequencies
