Multiple Focal Points

Use Frequency as Focal Point rather than one global gravitation.

This experiment tries to solve the issues overlapping links and nodes. The code is still using D3 and its Force Layout, but rather than having one central gravitational force every frequency node acts a focal point.

The concept of multiple focal points seem to work quite well, although there is still a problem of overlapping nodes. This seems to be more of an issue with D3's implementation and ideas than the concept of multiple focal points. As the experiment Horizontally Centered Layout provided more promising results, this approach was dropped.
